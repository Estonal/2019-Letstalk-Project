Letstalk project

2019년 6월 21일 제작했던 간단한 채팅 프로그램

1인 프로젝트

쓰레드를 이용하여 서버 하나에 여러 클라이언트가 접속할 수 있도록 하였음.
안타깝지만 완성도는 미흡함. 단기간 내 제출해야 했으며, 통신, GUI 관련 공부가 매우 미흡하여 독학해야 할 양이 많아 핵심적인 기능만 갖추게 하였음.


이 프로그램에서 제공하는 기능은 다음과 같음.

--클라이언트--
서버 IP를 통한 서버 접속 기능
해당 서버에 동시 접속해 있는 다른 클라이언트 리스트 확인 기능
전체 채팅 송수신 기능
채팅 명령어 기능(/quit, /to) : 각각 퇴장, 귓속말 기능

--서버--
서버 개방 기능
클라이언트 리스트 확인 기능
모든 클라이언트의 전체 채팅 수신기능


클라이언트 - Letstalk_Client
서버 - Letstalk_Server

Thread, Stream, Buffer, Socket, Synchronized 에 대해 스스로 공부하여 제작하였음.
GUI는 SceneBuilder를 이용하여 직접 제작하였음.

간단한 채팅 프로그램 하나를 제작하는 데에도 매우 어려움을 느꼈음.
